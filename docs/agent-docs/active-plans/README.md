# Active Plans (Placeholders)

This folder is reserved for **active/in-progress plans** that are being developed by agent-orchestrated workflows.

## Current Location
Active development plans currently reside in `pygeoinf/plans/` at the repository root. Once a plan is completed and approved, its phase-complete summaries graduate to `../completed-plans/`.

## Future Workflow
As agent tooling matures, this folder may contain "working drafts" or milestones from long-running projects. For now:

- **In-progress plans** → stay in root `plans/` folder (for visibility during development)
- **Completed plans** → archive to `../completed-plans/`
- **Research notes** → organize to `../references/`
